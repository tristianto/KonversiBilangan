# KonversiBilangan
Konversi Angka Menjadi Bilangan menggunakan Springboot

How To Use
- in main folder you can find 4 class java to run this application using Spring Tool Suite
- use Web, JPA, and H2 dependency using Spring Starter Project
- build the JAR file using Maven Build
- find the jar file in target folder in your project folder
